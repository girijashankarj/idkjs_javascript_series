Adds an element to the end of an array.

MDN Documentation

2. pop()

Removes and returns the last element of an array.

3. shift()

Removes and returns the first element of an array.

MDN Documentation

4. unshift(element)

Function Prototype Methods

1. apply(thisArg, argsArray)

Calls a function with a given this value and arguments provided as an array.

function greet(name) {    return `Hello, ${name}!`;}const result = greet.apply(null, ['Alice']); // Expected output: "Hello, Alice!"

MDN Documentation

2. call(thisArg, arg1, arg2, ...)

Calls a function with a given this value and individual arguments.

const result = greet.call(null, 'Bob'); // Expected output: "Hello, Bob!"

MDN Documentation

3. bind(thisArg, arg1, arg2, ...)

Creates a new function with a given this value and arguments.

const boundGreet = greet.bind(null, 'Charlie');const result = boundGreet(); // Expected output: "Hello, Charlie!"

MDN Documentation
